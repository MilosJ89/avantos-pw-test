## End to end tests suite

#### First time installation

1 - You need npm manager to be installed on your device

2 - Clone the project

3 - Open the terminal and type following commands:

`npm install` --> To install dependencies

`npx playwright install` --> To install browser drivers

#### Running tests headless locally - regression set
`npm run test:regression`

#### Running tests ui mode - regression set
`npm run test:regression:open`

#### Show report
`npm run show:report`
